1. app.js - запускаемый файл
  — список активных рекламных кампаний
  — запуск правил

2. ./lib/rules.js - правила
  — есть два правила: cutSources и cutCreatives
  — проверка по правилам,
    создание блэклиста,
    совершение нужного действия в источнике 
    трафика по этому листу,
    проставление значков на блэклиста

3.1. ./lib/binom.js - трекинг софт (база данных статистики)
    
3.2. ./lib/evadav.js - пример источника трафика
    в виде функционального программирования 
    основные действия с истоником трафика. 
    rules.js обобщает дейтсвия в унифицированный интерфейс,
    а тут уже более индивидуально (тк API у всех разные)
